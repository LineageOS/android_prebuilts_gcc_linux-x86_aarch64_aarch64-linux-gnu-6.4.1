  • Release Notes

The Linaro Toolchain Working Group (TCWG) is pleased to announce the 2017.08 stable Release of the Linaro GCC 6 source package.

This stable1 Release is based on FSF GCC 6.4.1+svn250045 and includes performance improvements and bug fixes backported from mainline GCC. This Release is based on Linaro Snapshot GCC 6.4-2017.08-rc1.

Changes in this stable GCC source package Release include:

  • Updates merged from FSF GCC 6.4.1+svn250045
  • Backport of [Misc] Regenerate intl configure
  • Backport of [AArch64] Fix -fstack-check with really big frames on aarch64
  • Backport of [Testsuite] Fix stack-check-1.c
  • Backport of [Testsuite] Add dg-require-stack-check
