This directory contains the GCC 6.4-2017.08 release of Linaro binary toolchain
as well as its source release package. This code is distributed under the
license terms specified in the accompanying files COPYING3 and COPYING.RUNTIME.

The binary can be built from the source package with instructions on Linaro
website: https://wiki.linaro.org/ABE. For your convenience, the relevant
information on that web page is saved in a file "Build.html" in this
directory.
